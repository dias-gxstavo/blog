## Introdução
Recentemente pude tirar um tempo para configurar meu servidor caseiro com calma e escolhi utilizar o [Homarr](https://homarr.dev/) para isso. 
O Homarr é um software **open-source** que oferece uma interface gráfica no estilo de dashboard para controlarmos o nosso servidor, o que reduz a necessidade do uso massivo de **CLI/YAML**.

![homarr](https://minio.gudilab.online/my-other-bucket/homarr-env/2026-01-25-114656_hyprshot.png)

## Aplicações via Docker

Uma das maiores vantagens do Homarr é que esse software é totalmente voltado ao ecossistema Docker. Podemos baixar milhares de serviços, também chamados de **Apps**, que nada mais são que imagens das suas respectivas aplicações. 

![appstore](https://minio.gudilab.online/my-other-bucket/homarr-env/2026-01-25-111001_hyprshot.png)

## Encontrando serviços de self-hosting

O Self-hosting é a pratica de hospedar e gerenciar aplicativos no seu próprio aplicativo invés de consumir serviços de terceiros

Para isso, utilizo o [linuxserver.io](https://www.linuxserver.io/our-images) e o [Awesome-Selfhosted](https://awesome-selfhosted.net/index.html).
Estes dois são grandes repositórios com diversos serviços, além de serem **open-source**.
